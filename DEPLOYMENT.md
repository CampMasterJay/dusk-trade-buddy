## Capacitor APK wrapping

Use the production URL as the remote source so Cloudflare serves the live build (no rebuild needed when the web app updates):

```bash
npm i -g @capacitor/cli
npx cap init "EdgeTrader" "com.edgetrader.app" --web-dir=dist
npx cap add android
```

Then in `capacitor.config.ts`:

```ts
import type { CapacitorConfig } from '@capacitor/cli';

const config: CapacitorConfig = {
  appId: 'com.edgetrader.app',
  appName: 'EdgeTrader',
  webDir: 'dist',
  server: {
    url: 'https://project--40f9eda6-08cc-49ab-a660-9e9f0cb6ba04.lovable.app',
    cleartext: false,
  },
  android: {
    backgroundColor: '#07070d',
  },
};

export default config;
```

Then:

```bash
npx cap sync android
npx cap open android   # Build → Generate Signed APK / Bundle in Android Studio
```

The native splash screen uses the same `#07070d` background and `icon-512.png`, matching the in-app `SplashScreen` component for a seamless launch.
